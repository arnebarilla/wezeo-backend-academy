# Wezeo Backend Academy
## PHP
### L1 : Logger 1
An application for logging student arrivals at school. Every time the script is run, the current time is recorded, saved to a JSON file, and all previous arrivals are displayed.

**How to run:**
1. Install [MAMP](https://www.mamp.info/)
2. Copy the `l1-logger` folder into `MAMP/htdocs/`
3. Start MAMP
4. Open browser and navigate to `http://localhost/l1-logger/logger1.php`
